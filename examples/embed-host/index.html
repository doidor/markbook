Markbook embed examples

Two ways to consume a Markbook story externally — both demos use the Pixie Button / Variants story from examples/react-demo.

  • Embed mode — drop in a <script type="module"> tag. Self-contained ESM, no host-side build.
  • Package mode — npm-style import via importmap, framework provided by the host (~3.5 KB story bundle).
  • Shadow-DOM isolation — package mode + mount(el, { isolation: 'shadow' }). Host-page CSS does not leak into the story.

Setup

Generate every bundle in one shot:

pnpm example:embed-host:build

Or, individually:

pnpm example:bundle           # embed mode (all stories)
pnpm example:bundle:package   # package mode (all stories)

Serve

pnpm example:embed-host:serve   # http://localhost:4500/embed-host/

The serve script roots Python's http.server at examples/ so the host can resolve ../react-demo/dist/... paths.
