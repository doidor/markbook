Embed mode

A Markbook story embedded externally with two lines of HTML — no iframe, no host-side build.

Setup

cd examples/react-demo
markbook bundle

Generates ../react-demo/dist/embed/components-button-variants.js. Each bundle is a self-contained ESM module with React, Pixie's Button, and the decorator stack inlined (~200 KB minified).

Markup

<div data-markbook-embed="components-button-variants"></div>
<script type="module" src="…/components-button-variants.js"></script>

The script auto-mounts on every matching <div> at DOMContentLoaded.
